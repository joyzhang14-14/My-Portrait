# Learned how agentic search and ReAct work

The user read an explanation of how models decide what to search for during multi-step reasoning. The discussion broke the process into world-knowledge priors, context-based next-token reasoning, and training on agentic trajectories with instruction tuning and reinforcement learning. It used The Count of Monte Cristo as an example, noting that a model might already remember the corpse-bag escape trick and that on a truly unfamiliar book it could fail to generate a query about how a prison handles bodies. The user also viewed a proposal to inspect a real ReAct trace step by step.
